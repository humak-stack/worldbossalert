# WorldBossAlert
A TurtleWoW addon for world boss scouting, alerts, and raid logging.

## Features
- Scout mode with clock-in/out and 30 min heartbeat
- Boss alerts to custom channel with @mentions
- Raid kill and loot logging with attendance (chunked for 255 char limit)
- Low level scout support (routes via officer/guild chat)
- Persistent GUI with minimap button

## Slash Commands
/wba scout — toggle scout mode
/wba raid  — toggle raid mode
/wba main <name> — set your main name
/wba help  — show all commands

## Installation
1. Download and unzip
2. Place the WorldBossAlert folder in: WoW/Interface/AddOns/
3. Reload UI or log in
